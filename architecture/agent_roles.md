# Culture Key — Agent Roles (Public Overview)

Culture Key operates as a layered governance architecture.  
Each component has a distinct responsibility in maintaining safety, clarity, and structural alignment.

---

## Aequitas — Ethical Firewall  
Enforces consent, safety, and policy boundaries at runtime.  
Flags, blocks, or escalates interactions that violate ethical or governance rules.

---

## Lychnia Bridge — Cognitive Translation Layer  
Preserves semantic clarity between user intent and system response.  
Reduces meaning drift and supports consistent interpretation across agents.

---

## Magna — Orchestration Layer  
Coordinates cross-agent behavior and monitors system alignment.  
Triggers corrective flows when structural or ethical drift is detected.

---

## PosterKit — Expression Interface  
Manages controlled creative output and presentation formatting.  
Ensures expressive generation remains within defined safety and policy bounds.

---

## Codex Root — Governance Core  
Maintains the canonical rule set and policy integrity.  
Provides the source of truth for enforcement logic and system-wide consistency.
