# ⚙️ Ethical State Machine — High-Level View

The Ethical State Machine is the high-level mechanism that ensures
AI behaviour remains aligned with human values.

### Public States
- **SAFE** — everything is normal  
- **REVIEW** — ambiguous or sensitive input  
- **BLOCKED** — clear ethical violation  
- **ESCALATE_TO_HUMAN** — requires human judgment  

This is the *outer layer* of the system.
All internal transitions, rules, checks and enforcement logic
remain private inside the Culture Key OS.

The purpose of this public diagram is clarity:
to show that the ecosystem operates with structure and predictable logic.

See State Machine Rules for transition logic.

# GR ⚙️ Ethical State Machine — High-Level View

# Η Ethical State Machine είναι ο μηχανισμός που διασφαλίζει ότι η 
συμπεριφορά της ΤΝ παραμένει ευθυγραμμισμένη με τις ανθρώπινες αξίες.

### Δημόσιες Καταστάσεις
- **SAFE** — φυσιολογική λειτουργία  
- **REVIEW** — ασαφές ή ευαίσθητο περιεχόμενο  
- **BLOCKED** — σαφής ηθική παραβίαση  
- **ESCALATE_TO_HUMAN** — απαιτεί ανθρώπινη κρίση  

Πρόκειται για το *εξωτερικό επίπεδο* του συστήματος.  
Όλοι οι εσωτερικοί κανόνες, μεταβάσεις και οι μηχανισμοί ελέγχου 
παραμένουν ιδιωτικοί.

Ο σκοπός της δημόσιας παρουσίασης είναι η διαφάνεια και η δομημένη εικόνα.
