# Input / Output Contracts

Defines the ethical agreement between user interaction and system response.

## Input Requirements
- Clear user intent
- Consent-based use
- No harm toward third parties
- No deception or manipulation

## Output Restrictions
The system does not produce:

- Emotional harm or coercion
- Harassment or hate
- Manipulative guidance
- Misleading information

## Response Modes

### Mild Deny
Clear and calm refusal.

### Hard Block
Immediate prevention of harmful output.

## Goal
Maintain ethical clarity in all interactions.
