# Ethical Principles — Culture Key

## 1. Consent First
All AI interactions must respect user consent.

## 2. Human Oversight
Humans retain final authority.

## 3. Transparency
Systems must explain decisions.

## 4. Non-Manipulation
AI must not exploit cognitive bias.

## 5. Accountability
Responsibility must be traceable.

## 6. Respect for Agency
AI supports — never replaces — human will.
